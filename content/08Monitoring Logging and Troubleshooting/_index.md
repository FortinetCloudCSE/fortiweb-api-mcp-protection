## **Objective**

Investigate events and troubleshoot security issues using FortiWeb logs.

### **Topics Covered**

#### **Logging Fundamentals**

* Attack logs  
* Event logs  
* Traffic logs  
* System logs

#### **Troubleshooting Methodology**

* Verify traffic flow  
* Identify blocked requests  
* Correlate events  
* Validate policies
