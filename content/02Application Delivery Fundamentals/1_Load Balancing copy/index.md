#### **Load Balancing**  (This is a review only section, everything is preconfigured)

LAB Define Applications to Be Protected
Assuming you are finished with typical housekeeping items , such as defining network interfaces, DNS Server, Admin users etc. associated with a network appliance the first step with configuration of Fortiweb is to define the application that will be protected by Fortiweb via a server pool . A Server Pool may contain a single server or multiple servers. If you add more than one server in a Server Pool, you will have the opportunity to configure additional options that are found in a typical load balancer such as health checks, load balancing algorithm, and more.

In our Lab you will notice that we have defined multiple pools. We will take a look at one of the pools and examine the details.
