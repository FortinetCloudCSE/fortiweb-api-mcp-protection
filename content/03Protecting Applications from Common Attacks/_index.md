## **Objective**

Learn how FortiWeb identifies and blocks common web application attacks by first observing an unprotected application, then applying a Web Protection Profile and reviewing the resulting attack detections.

### **Topics Covered**

#### **Web Application Threats**

* SQL Injection
* Cross-Site Scripting (XSS)
* Command Injection
* Path Traversal
* Protocol Violations

#### **Security Policies**

* Web Protection Profiles
* Attack Signatures
* HTTP Protocol Constraints
* Client-side and cookie security

### **Key Takeaways**

* Understand how common application attacks behave when no WAF is present
* Learn how a Web Protection Profile groups FortiWeb security controls
* Observe how FortiWeb detects, categorizes, logs, and blocks malicious requests
